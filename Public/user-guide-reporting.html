PDF Bundle Summary

Problem: Not knowing page counts or sizes across thousands of PDFs.

Solution: Generates a concise Excel summary ("borderou") with names, sizes, and verified page counts.

  • Instant total page count for printing/costing.
  • Detects unreadable or corrupt PDFs.
  • Export to XLSX for easy spreadsheet management.
📑

Advanced Bundle Report

Problem: Need for deep metadata (Dates, Authors) across various formats.

Solution: An enhanced report covering PDF, Word, and Excel with extended timeline metadata.

  • Captures Creation and Modification dates.
  • Smart page counting for Office formats.
  • Detailed logical path mapping.
📁

Folder Evidence Report

Problem: Mixed media trees (Audio, Video, PDF) that need a full inventory.

Solution: Exhaustive inventory of every file in a tree, calculating hashes and tree depths.

  • Maps entire workstation architecture.
  • MD5/SHA256 hash generation for forensic integrity.
  • Identifies hidden or system files in evidence sets.
